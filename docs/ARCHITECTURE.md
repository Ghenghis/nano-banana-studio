## Security Considerations

1. **API Keys**: All secrets stored in `.env`, never in code
2. **CORS**: Configurable origin restrictions
3. **Rate Limiting**: Recommended for production deployment
4. **Input Validation**: Pydantic models validate all inputs
5. **File Upload**: Type and size restrictions enforced

## Performance Optimization

1. **Parallel Processing**: Multiple scenes generated concurrently
2. **Caching**: Fingerprint-based deduplication
3. **Streaming**: WebSocket for real-time updates
4. **Queue System**: Redis-based job queue for heavy operations
5. **GPU Offloading**: Optional GPU worker for local AI models
